Key Bindings

Shift-A Treeline tracing mode
Shift-S Synapse dropping mode
Shift Join active node to a target node
T Tag active treenode or connector
+ Zoom in
- Zoom out
, z dimension up
. z dimension down
Shift-Doubleclick on node Zoom in on node
Ctrl-Click empty space Deactive currently active node
Click empty space in synapse dropping mode Create new connector
Click empty space in tracing mode Create new treenode (when no node active)
Click empty space in tracing mode Create new child treenode (with active treenode)
